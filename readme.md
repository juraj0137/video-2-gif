# Video 2 gif

## Installation

Clone this repo. 
`git clone https://github.com/juraj0137/video-2-gif.git video-2-gif`

Go to cloned folder
`cd video-2-gif`

Remove git folder, it is not necessary here
`rm -rf .git`

Make `install.sh` executable 
`sudo chmod +x ./install.sh`

Install dependecies
`./install.sh`
